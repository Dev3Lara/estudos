JS- Scripts básicos e comentários

Para programar em js usamos a tag <script> dentro do body

E para comentar numa linha //. Ou, para comentar de um trecho até outro, abrimos o comentário com /* e fechamos com */

Scripts e comentários 

                
            < script >    
                window.alert('Exemplo de alerta')
                    //Exibir caixa de alerta
                window.confirm('Exemplo de cancelamento ou confirmação')
                    //Caixa pedindo confirmação ou não
                window.prompt('Exemplo de promt (pergunta)')
                    /* Input de dados (pergunta) */

            < /script >

Pra que servem os comentários em js?

Para dizer o que aquele código faz. É como se explicasse o porquê aquele código está ali. Não a função dele em si no js, mas o porquê você invocou aquela função naquele momento. Assim, outro Dev, ou você mesmo anos depois, vai conseguir compreender o código e fazer upgrades com facilidade.

Então, SEMPRE comnente (Legende) seu código!
